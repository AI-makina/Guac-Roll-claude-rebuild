# Guac & Roll - Mexican Flavor on Tour

A modern, responsive website for the Guac & Roll food truck featuring an online ordering system.

## Quick Start

### Option 1: Open directly
Simply open `index.html` in your web browser.

### Option 2: Local server (recommended)
```bash
# Using Python 3
cd guac&roll_claude
python3 -m http.server 8000

# Then visit http://localhost:8000
```

### Option 3: VS Code Live Server
If using VS Code, install the "Live Server" extension and click "Go Live".

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop
- **Online Ordering**: Add items to cart, adjust quantities, checkout
- **Menu Categories**: Tacos, Burritos, Bowls, Sides, Drinks
- **Shopping Cart**: Persistent cart with tax and tip calculation
- **Nutrition Info**: View nutritional details for each item
- **Smooth Animations**: Hero CTA animation, card entrance effects
- **Contact Section**: Hours, location, phone, email
